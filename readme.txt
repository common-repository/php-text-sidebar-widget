== Description ==

Simple SideBar PHP enabled Widget. Specify and execute arbitrary PHP code as your sidebar block.

== Installation ==

1. Upload `wp-phptext.php` to the `/wp-content/plugins/wp-phptext/wp-phptext.php` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add a new widget to your sidebar in Design section
